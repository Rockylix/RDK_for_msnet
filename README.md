# 项目说明

本项目主要包含三个功能模块，分别对应三个文件夹，**其中每个模块都有自己的log开发说明**：

## 1. capture
- **功能**：MIPI 摄像头 API 调用测试。
- **说明**：用于测试和验证 MIPI 摄像头的底层 API 调用及图像采集功能。

## 2. msnet
- **功能**：图片测试网络。
- **说明**：基于静态图片输入的深度学习网络推理测试模块。

## 3. mipi_msnet
- **功能**：实时采集与网络推理结合。
- **说明**：将 MIPI 摄像头实时采集的数据直接送入网络进行推理。
- **特性**：该模块支持配置是否开启可视化显示。
